# WCSU Archives and Special Collections

This organization hosts open-source tools and infrastructure developed by the **Western Connecticut State University Archives and Special Collections** to support archival discovery and digital access.

## About WCSU Archives

The WCSU Archives, established in 1977, maintains the historical records of WestConn dating back to its founding in 1903 as the Danbury Normal School. Our holdings include approximately 1,800 linear feet of archival materials and 9,000 cataloged titles in Special Collections, covering local history, university records, rare books, and various media formats.

**Website:** [https://archives-library.wcsu.edu](https://archives-library.wcsu.edu)

## Repositories

### [wcsu-scripts](https://github.com/WCSUArchives/wcsu-scripts)
A collection of shell scripts for automating EAD (Encoded Archival Description) indexing and maintenance in Arclight. These utilities handle:
- Automatic detection and re-indexing of EAD changes
- Monitoring for deleted files and queueing SOLR removals
- Repository setup and directory structure management

### [wcsu-arclight](https://github.com/WCSUArchives/wcsu-arclight)
Setup and configuration documentation for deploying the [Arclight](https://github.com/projectblacklight/arclight) archival discovery platform at WCSU. Includes instructions for:
- Ruby/Rails environment setup
- Subdirectory deployment configuration
- Apache proxy configuration
Also includes customizations like
- Custom templates
- Well, mostly templates
